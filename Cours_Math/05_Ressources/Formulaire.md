# Formulaire de Mathématiques Ultime

L'antisèche absolue de l'élève. Voici les formules principales à apprendre par cœur lors des études, regroupées par grands thèmes.

---

## 1. Mémorandum Géométrie

### Aires des figures Planes
- **Carré** : $c \times c$ ou $c^2$
- **Rectangle** : $L \times l$
- **Triangle** : $\frac{\text{Base} \times \text{Hauteur}}{2}$
- **Disque** : $\pi \times R^2$ *(Attention : Le périmètre du cercle c'est $2 \pi R$ !)*
- **Parallélogramme** : $\text{Base} \times \text{hauteur}$

### Volumes des Solides
- **Cube** : $c \times c \times c = c^3$
- **Pavé (Parallélépipède)** : $L \times l \times h$
- **Prisme / Cylindre droit** : $\text{Aire de la base} \times h$
- **Pyramide / Cône ("Pointus")** : $\frac{\text{Aire de la base} \times h}{3}$
- **Boule** : $\frac{4}{3} \pi R^3$

---

## 2. Géométrie dans les Triangles

### Théorème de Pythagore (Triangle rectangle)
- Si le triangle $ABC$ est rectangle en $A$ : $\mathbf{BC^2 = AB^2 + AC^2}$ *(L'hypoténuse au carré équivaut à la somme des carrés des autres côtés)*.

### Trigonométrie (SOH CAH TOA)
- $\sin(\alpha) = \frac{\text{Opposé}}{\text{Hypoténuse}}$
- $\cos(\alpha) = \frac{\text{Adjacent}}{\text{Hypoténuse}}$
- $\tan(\alpha) = \frac{\text{Opposé}}{\text{Adjacent}}$
- Propriété d'or : $\cos^2(x) + \sin^2(x) = 1$

### Théorème de Thalès (Droites parallèles)
- Si $(MN)$ est parallèle à $(BC)$ : $\mathbf{\frac{AM}{AB} = \frac{AN}{AC} = \frac{MN}{BC}}$.

## 3. Algèbre et Calcul Formel

### Puissances
- $x^a \times x^b = x^{a+b}$
- $\frac{x^a}{x^b} = x^{a-b}$
- $(x^a)^b = x^{a \times b}$

### Identités Remarquables
- $(a+b)^2 = a^2 + 2ab + b^2$
- $(a-b)^2 = a^2 - 2ab + b^2$
- $(a-b)(a+b) = a^2 - b^2$ *(Incontournable pour tuer des racines carrées au dénominateur !)*

### Solutions du Second Degré ($ax^2 + bx + c = 0$)
1. Le $\Delta$ (Discriminant) : **$\Delta = b^2 - 4ac$**
2. Si $\Delta > 0$, deux racines (solutions) réelles :
   $x_1 = \frac{-b - \sqrt{\Delta}}{2a}$ \qquad \text{et} \qquad $x_2 = \frac{-b + \sqrt{\Delta}}{2a}$
3. Si $\Delta = 0$, une racine double : $x_0 = \frac{-b}{2a}$

### Logarithme et Exponentielles
- $\ln(a \times b) = \ln(a) + \ln(b)$
- $\ln(a/b) = \ln(a) - \ln(b)$
- $\ln(a^n) = n\ln(a)$
- $\ln(e) = 1$ et $e^0 = 1$

## 4. Analyse et Dérivées

### Formules de Dérivation Opérationnelle
- Somme : $(u+v)' = u' + v'$
- Produit : $(u \times v)' = \mathbf{u'v + uv'}$ *(Le grand piège! Le produit des dérivées est FAUX)*
- Quotient : $\left(\frac{u}{v}\right)' = \mathbf{\frac{u'v - uv'}{v^2}}$

### Dérivées Usuelles
| Fonction $f(x)$ | Dérivée $f'(x)$ |
| :---: | :---: |
| $k$ (constante) | $0$ |
| $x$ | $1$ |
| $x^2$ | $2x$ |
| $x^n$ | $nx^{n-1}$ |
| $\frac{1}{x}$ | $-\frac{1}{x^2}$ |
| $\sqrt{x}$ | $\frac{1}{2\sqrt{x}}$ |
| $e^x$ | $e^x$ |
| $\ln(x)$ | $\frac{1}{x}$ |

---

## 5. Probabilités

- Formule Totale / Intersection de base : **$P(A \cup B) = P(A) + P(B) - P(A \cap B)$**
- Probabilité Conditionnelle : **$P_A(B) = \frac{P(A \cap B)}{P(A)}$**
- Indépendance : $P(A \cap B) = P(A) \times P(B)$
- Loi Binomiale $\mathcal{B}(n, p)$ : Espérance $E(X) = n \times p$
- Loi Uniforme sur $[a;b]$ : $E(X) = \frac{a+b}{2}$
- Loi Exponentielle $\mathcal{E}(\lambda)$ : $E(X) = \frac{1}{\lambda}$
